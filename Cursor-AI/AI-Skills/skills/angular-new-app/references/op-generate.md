# Operation: Angular-Artefakte generieren

## Trigger-Keywords
`generate`, `ng generate`, `Komponente erstellen`, `Service erstellen`, `neues Artefakt`

---

## Schritte

### Code-Generierung
```
npx ng generate component|service|pipe|directive|interface|guard|interceptor|resolver|enum|class <name>
```

- Rückgegebenen Pfad merken.
- CLI generieren → dann Code für App-Anforderungen anpassen.

---

_IMPORTANT:_ Best Practices: `npx ng mcp` → `get_best_practices`.
